
-------------------
Installation steps:
-------------------

Please follow the steps to install the Cardinity Payment Module on your shop

1) Copy the sub folders of "copy_this" folder contents into the root path of your shop.

2) Please execute the sql code from the "Install" file

3) Activate the payment module by going to:

   Admin -> Extensions -> Modules -> Cardinity

   In the overview tab please click "Activate" button to activate it and logoff from the admin area.

4) Login again in the admin area:
    - Go to the Cardinity Config link in left panel under Cardinity.
    - Enter the necessary Cardinity Parameters required for the payment.

5) Clear the folder contents in "[rootpath]/tmp"
